# Rituvo

Rituvo is a social accountability product for helping people build habits, reach goals, and reduce procrastination through commitment to other people.

The project documentation is designed to provide coding agents and contributors with concise, retrieval-friendly context.

Start with:

- [`AGENTS.md`](AGENTS.md)
- [`docs/README.md`](docs/README.md)

The current implementation priority is a believable first demo, not a complete production platform.
